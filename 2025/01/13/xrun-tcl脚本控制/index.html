Xcelium仿真器命令语言是基于TCL，能够对模拟器中的对象执行各种操作。

The Tcl Command-Line Interface

Tcl Command Syntax

Tcl Command Rules

  • 命令的名称必须小写

  • 可以使用略缩表达

  • 一行可以有多个tcl命令，使用;分割

Command Description Conventions

  • 格式符号 的作用是规范 Tcl 命令的书写和使用，方便用户理解命令中的必填和可选部分。
  • 核心规则：
    1. 必须输入的部分: 使用字面量。
    2. 可选部分: 用方括号 [ ] 标注。
    3. 选择其中一个: 用竖线 | 和大括号 { } 表示。
    4. 重复参数: 用省略号 ... 表示，方括号则标明是否可以完全省略。

通过这些符号的配合，Tcl 命令的表达既清晰又灵活，适应多种使用场景。

database
