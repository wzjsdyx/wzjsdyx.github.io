内存空间查询

详细的report

ps：process status显示当前运行中的进程

-u $USER：只列出当前用户的进程

-o pid,comm,rss：只输出这三列

  • pid：进程 ID
  • comm：命令名称（程序名）
  • rss：实际使用的物理内存（单位：KB)

--sort=-rss：按内存使用量从大到小排序

awk：文本处理工具

磁盘空间查询

用户磁盘配额

quota命令用于显示用户的磁盘使用量和限制

当前目录磁盘占用空间

这个命令的作用是：

  • du: 是 disk usage 的缩写，用于查看目录或文件所占用的磁盘空间大小。
  • -h: 以人类可读的格式显示文件大小，例如 GB、MB 等。
  • --max-depth=1: 限制显示的深度，即只显示当前目录下直接子目录的信息。

运行这个命令后，会列出当前目录下每个子目录的总大小，并以人类可读的方式显示（如 GB、MB 等）。

按大小排序显示

sort -h：按人类可读的格式排序，从小到大排列

文件名查找

管道命令

grep "搜索内容" 文件名：这是用于搜索的命令。你可以替换 "搜索内容" 和 文件名 为你想要的内容和文件。

|：管道符，将前一个命令的输出传递给下一个命令。

tee 搜索结果.txt：tee 命令会将接收到的输出同时写入 搜索结果.txt 文件并显示在屏幕上。
