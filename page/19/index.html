工艺库-基本概念

发表于 2024-04-02 更新于 2025-04-03 分类于 工艺库 Valine：

阈值电压HVT/RVT/LVT PPA对比

NOTE:

实际使用的时候，LVT和HVT的stand cell的面积几乎没什么区别；

之所以说使用LVT综合出来的面积比HVT综合出来的面积小，是因为一个LVT在驱动能力上可能比几个HVT好，所以用LVT之后，整体的cell数量就会降低；

工艺角SS/TT/FF和片上误差OCV

由于半导体制造过程中存在工艺波动，相同设计的芯片在不同批次中会有性能差异。

  • 工艺角用来表示芯片在不同制造条件下的性能变化范围。
  • 常用的工艺角有：
    • SS：Slow-Slow
    • FF：Fast-Fast
    • TT：Typical-Typical
    • SF/FS：Slow-Fast / Fast-Slow（P管和N管速度不匹配）

同一个批次的wafer其corner是相同的，例如下订单之后，其所有的wafer都是相同的corner；

然后同一片wafer上，不同的位置，其性能有可能也是不同的，这就是OCV的概念；

OCV的全称是On Chip Variation，用于描述在一块芯片上不同管子间由于工艺偏差、电压降以及温度变化（PVT）等引起的delay变化

PMK

是低功耗相关的库，例如ISO/power switch，level shift等

栅极长度

HDC50和HDC40

数字表示栅极长度；

沟道长度越短，速度越快，功耗越高

Track

1. Track的含义

  • 基本定义：
    • 一个Track通常表示一个金属布线层的宽度和间距之和，即布线间隔的单位长度。
    • 单元高度通常是以Track数来表示的，例如8-Track、9-Track、12-Track等。
  • 标准单元高度：
    • 单元高度 = Track数 × 单一轨道高度。
    • 一般情况下，1个Track约等于工艺栅极长度（Gate Length），比如40nm工艺中，1个Track大约是40nm。

4. Track对PPA的影响

Track数 面积（Area） 性能（Performance） 功耗（Power）
低Track 面积小，密度高 性能较低 功耗低
高Track 面积大，密度低 性能较高 功耗高

分析：

  • 低Track数（如8-Track）：
    • 面积小：标准单元较矮，布局密度高。
    • 性能低：单元较矮，驱动能力受限，不适合高速逻辑。
    • 功耗低：由于短导线和小驱动器，功耗较低。
  • 高Track数（如12-Track）：
    • 面积大：标准单元较高，导致芯片面积增大。
    • 性能高：更大的单元高度允许更强的驱动能力，有助于高频电路。
    • 功耗高：驱动能力增强的同时，漏电流和动态功耗增加。

如下所示是上述运算的FPGA综合原理图，其中LUT是不同的逻辑函数：实现的是译码器+多路选择器

移位逻辑本质上是译码逻辑和多路选择逻辑

CPU内部的移位操作实现方式

CPU内部存在逻辑左移，逻辑右移，算数右移三种移位操作，最直观的方式是直接使用<< >> >>>，最后三个结果根据运算符进行三选一mux操作输出。

但是通过上面的移位运算的实现原理知道，这样实现的方式面积大消耗比较大

这里采用面积优化的移位逻辑实现。

  • 逻辑左移实现
    • 逆序排列
    • 左移即可变成右移
    • 再逆序得到结果
  • 逻辑右移
    • 直接右移得到结果
  • 算数右移
    • 先右移
    • 高位补充符号位

思路是，所以的移位操作只使用一个移位运算符。

上面的实现方式只有一个shft_res逻辑是一个完整的32位移位操作（被移位的数值和移位量都是变量）；sra_mask的逻辑被移位的数值是常量，综合工具会做优化，因此实际消耗的逻辑资源远远少于一个完整的32位移位操作。缺点是时序延迟会有所增加。

基础知识-有符号移位和无符号移位

发表于 2024-04-01 更新于 2024-04-08 分类于 CPU设计实战 Valine：

实际的移位运算中，不区分是有符号数移位和无符号移位，而是通过算术移位和逻辑移位来表示移位运算

–》有符号数和无符号数都可以使用算术移位和逻辑移位

逻辑左移和算术左移

作用是相同的，都是在最低位置补充0；

MIPS指令集没有算术左移，因为没对逻辑左移和算术左移做特殊区分

其他指令集有算术左移，是考虑了指令的一致性和完整性

ro环形振荡电路

特殊的设计中，可能还会有组合逻辑回环（例如TRNG的环形振荡器电路）

仿真惊奇地停滞在：200ns(参考testbench延时)

在环形振荡器原理图中，假设第一级反相器的输入端为低电平，经过5级反相之后，第5级反相器输出端必为高电平。在t=0时刻，此时电源(VDD)通过第5级反相器输出端对第1级反相器的输入电容充电，第一级反相器经过一级平均延迟时间Tdelay1之后，其输出状态发生转换。经过(Tdelay1+ Tdelay2+ Tdelay3+ Tdelay4+ Tdelay5)之后，第5级反相器输出端由高电平转换为低电平。接着第一级的输入电容将通过第5级的输出端放电，经过(Tdelay1+ Tdelay2+ Tdelay3+ Tdelay4+ Tdelay5)之后状态再次发生转换，完成一个振荡周期输出，如下图的方波信号。如果反相器各级的平均延迟时间相等，为Tdelay，

即：多个反相器组成环形振荡器的一个脉冲周期为T=10 Tdelay

如果想要不停滞，可以在任何一级加一个delay，例如u1和u2之间delay 5ns
