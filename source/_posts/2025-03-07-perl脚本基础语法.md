# GetOptions命令行参数解析

```perl
GetOptions(
    'inports=i'      => \$SlaveInterfaces,       # 读取 --inports 参数，值必须是整数，存入 $SlaveInterfaces
    'outports=i'     => \$MasterInterfaces,      # 读取 --outports 参数，值必须是整数，存入 $MasterInterfaces
    'connectivity=s' => \$Connectivity,          # 读取 --connectivity 参数，值必须是字符串，存入 $Connectivity
    'arb=s'          => \$ArbiterType,           # 读取 --arb 参数，值必须是字符串，存入 $ArbiterType
    'arch=s'         => \$ArchitectureType,      # 读取 --arch 参数，值必须是字符串，存入 $ArchitectureType
    'cfg=s'          => \$XmlConfigFile,         # 读取 --cfg 参数，值必须是字符串，存入 $XmlConfigFile
    'datawidth=i'    => \$RoutingDataWidth,      # 读取 --datawidth 参数，值必须是整数，存入 $RoutingDataWidth
    'addrwidth=i'    => \$RoutingAddressWidth,   # 读取 --addrwidth 参数，值必须是整数，存入 $RoutingAddressWidth
    'userwidth=i'    => \$UserSignalWidth,       # 读取 --userwidth 参数，值必须是整数，存入 $UserSignalWidth
    'outputarb=s'    => \$OutputArbName,         # 读取 --outputarb 参数，值必须是字符串，存入 $OutputArbName
    'outputstage=s'  => \$OutputStageName,       # 读取 --outputstage 参数，值必须是字符串，存入 $OutputStageName
    'matrixdecode=s' => \$MatrixDecodeName,      # 读取 --matrixdecode 参数，值必须是字符串，存入 $MatrixDecodeName
    'inputstage=s'   => \$InputStageName,        # 读取 --inputstage 参数，值必须是字符串，存入 $InputStageName
    'busmatrix=s'    => \$BusMatrixName,         # 读取 --busmatrix 参数，值必须是字符串，存入 $BusMatrixName
    'help'           => \$Help,                  # 读取 --help 参数，如果存在则 $Help 为真
    'verbose'        => \$Verbose,               # 读取 --verbose 参数，如果存在则 $Verbose 为真
    'srcdir=s'       => \$SourceDir,             # 读取 --srcdir 参数，值必须是字符串，存入 $SourceDir
    'tgtdir=s'       => \$TargetDir,             # 读取 --tgtdir 参数，值必须是字符串，存入 $TargetDir
    'xmldir=s'       => \$XmlDir,                # 读取 --xmldir 参数，值必须是字符串，存入 $XmlDir
    'overwrite'      => \$Overwrite,             # 读取 --overwrite 参数，如果存在则 $Overwrite 为真
    'notimescales'   => \$NoTimescales,          # 读取 --notimescales 参数，如果存在则 $NoTimescales 为真
    'check'          => \$CheckMode,             # 读取 --check 参数，如果存在则 $CheckMode 为真
    'spirit'         => \$Spirit                 # 读取 --spirit 参数，如果存在则 $Spirit 为真
);
```

> NOTE:
>
> - `=i`：表示该选项的值必须是整数。
> - `=s`：表示该选项的值必须是字符串。
> - 没有 `=` 的选项（如 `help`、`verbose`）：表示这是一个布尔选项，不需要值。如果用户在命令行中提供了该选项，则对应的变量会被设置为真（`1`），否则为假（`0`）。
