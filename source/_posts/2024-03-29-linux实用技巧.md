# 内存空间查询

## 详细的report

```bash
#!/bin/bash

# 获取当前用户
user=$USER

# 获取当前时间
echo "===== 当前用户内存使用报告 ====="
echo "用户: $user"
echo "时间: $(date)"

# 获取系统总物理内存（单位 GB）
total_phys_mem=$(free -g | awk '/^Mem:/ {print $2}')

# 当前用户物理/虚拟内存总和（GB）
read user_rss_gb user_vsz_gb <<< $(ps -u $user -o rss=,vsz= | \
  awk '{rss_sum+=$1; vsz_sum+=$2} END {printf "%.2f %.2f", rss_sum/1024/1024, vsz_sum/1024/1024}')

# 系统所有进程的虚拟内存总和（GB）
total_vsz_gb=$(ps -eo vsz= | awk '{vsz_sum+=$1} END {printf "%.2f", vsz_sum/1024/1024}')

echo
echo "===== 🧠 内存总览（单位 GB） ====="
echo "系统总物理内存:        $total_phys_mem GB"
echo "当前用户物理内存总和:  $user_rss_gb GB"
echo "当前用户虚拟内存总和:  $user_vsz_gb GB"
echo "系统全部虚拟内存总和:  $total_vsz_gb GB"
echo

# 打印表头
printf "%-8s %-20s %-12s %-12s\n" "PID" "COMMAND" "RSS(MB)" "VSZ(MB)"

# 打印当前用户每个进程的内存信息，按VSZ从大到小排序
ps -u $user -o pid,comm,rss,vsz --sort=-vsz | \
awk '
NR > 1 {
    rss_mb = $3 / 1024
    vsz_mb = $4 / 1024
    total_rss += rss_mb
    total_vsz += vsz_mb
    printf "%-8s %-20s %-12.2f %-12.2f\n", $1, $2, rss_mb, vsz_mb
}
END {
    print "------------------------------------------------------------"
    printf "%-8s %-20s %-12.2f %-12.2f\n", "TOTAL", "", total_rss, total_vsz
}'

```

> ---
>
> `ps`：process status显示当前运行中的进程
>
> `-u $USER`：只列出当前用户的进程
>
> `-o pid,comm,rss`：只输出这三列
>
> - `pid`：进程 ID
> - `comm`：命令名称（程序名）
> - `rss`：实际使用的物理内存（单位：KB)
>
> `--sort=-rss`：按内存使用量从大到小排序
>
> `awk`：文本处理工具





# 磁盘空间查询



## 用户磁盘配额

```shell
quota -s
# 查看配额并以易读的方式显示
```

`quota`命令用于显示用户的磁盘使用量和限制



## 当前目录磁盘占用空间

```shell
du -h --max-depth=1
```



> 这个命令的作用是：
>
> - `du`: 是 disk usage 的缩写，用于查看目录或文件所占用的磁盘空间大小。
> - `-h`: 以人类可读的格式显示文件大小，例如 GB、MB 等。
> - `--max-depth=1`: 限制显示的深度，即只显示当前目录下直接子目录的信息。
>
> 运行这个命令后，会列出当前目录下每个子目录的总大小，并以人类可读的方式显示（如 GB、MB 等）。



 **按大小排序显示**

```shell
du -sh * | sort -h
```

> `sort -h`：按人类可读的格式排序，从小到大排列

# 管道命令

```shell
grep "搜索内容" 文件名 | tee 搜索结果.txt
```

> `grep "搜索内容" 文件名`：这是用于搜索的命令。你可以替换 `"搜索内容"` 和 `文件名` 为你想要的内容和文件。
>
> `|`：管道符，将前一个命令的输出传递给下一个命令。
>
> `tee 搜索结果.txt`：`tee` 命令会将接收到的输出同时写入 `搜索结果.txt` 文件并显示在屏幕上。

# log打印

## 执行信息打印

### bash脚本

```shell
# 执行并记录
./tc.sh 2>&1 | tee tc.log

# 查看日志
cat tc.log

# 同时监控日志
tail -f tc.log
```

### tcsh脚本

```shell
# 执行脚本并将输出同时显示在屏幕和保存到文件
./tc.csh |& tee tc.log

# 追加模式
./tc.csh |& tee -a tc.log

# 带时间戳的日志
./tc.csh |& tee >(perl -ne 'print scalar(localtime), " ", $_') > tc.log
```
