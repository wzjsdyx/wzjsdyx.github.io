# 源代码分析



## sizeof

> `sizeof` 是 C 语言中的一个 **运算符**（不是函数），用于计算<font color=blue>变量或数据类型</font>在内存中占用的字节数。它的作用包括：
>
> - <font color=blue>**静态计算**</font>：在编译时确定结果，不会在运行时计算。
> - <font color=blue>**适用对象**</font>：
>   - 基本数据类型（如 `int`、`char`）。
>   - 数组（返回整个数组的字节数）。
>   - 指针（返回指针本身的字节数，而非指向的内容）。
>   - 结构体或联合体。
>   - 无法计算malloc，动态分配内存的大小。

## memset

```c
#include <string.h>  // 必须包含的头文件
void *memset(void *ptr, int value, size_t num);
参数：
ptr：指向要填充的内存块的指针（如数组、结构体或动态分配的内存）。
value：要设置的值（以 int 形式传入，但实际按 无符号字符 处理，范围 0 到 255）。
num：要填充的字节数。
```



## fwrite/fread

```shell
NAME
       fread, fwrite - binary stream input/output
SYNOPSIS
       #include <stdio.h>

       size_t fread(void *ptr, size_t size, size_t nmemb, FILE *stream);

       size_t fwrite(const void *ptr, size_t size, size_t nmemb, FILE *stream);

DESCRIPTION
       The function fread() reads nmemb items of data, each size bytes long, from the stream pointed to by stream, storing them at the location given by ptr.

       The function fwrite() writes nmemb items of data, each size bytes long, to the stream pointed to by stream, obtaining them from the location given by ptr.
    
```

void Database_close(struct Connection *conn,int MAX_ROWS)
{
    printf("Enter Database_close\n");
    if(conn) {

        int i = 0;

}

        printf("ERROR: %s\n", message);

struct Connection *Database_open(const char *filename, char mode, int MAX_ROWS, int MAX_DATA)
{
    printf("Enter struct Connection\n");
    // 内存申请
    struct Connection *conn = malloc(sizeof(struct Connection));
    if(!conn) die("Memory error", NULL);

    conn->db = malloc(sizeof(struct Database));
    if(!conn->db) die("Memory error", conn);

    conn->db->rows = malloc(sizeof(struct Address) * MAX_ROWS);
    if(conn->db->rows == NULL) die("Memory error", conn);

    int i = 0;
    for(i = 0; i < MAX_ROWS; i++){
        (conn->db->rows[i]).name = malloc(sizeof(char) * MAX_DATA);
        if((conn->db->rows[i]).name == NULL) die("Memory error", conn);
        (conn->db->rows[i]).email = malloc(sizeof(char) * MAX_DATA);
        if((conn->db->rows[i]).email == NULL) die("Memory error", conn);
    }

    if(mode == 'c') {
        conn->file = fopen(filename, "w");
    } else {
        conn->file = fopen(filename, "r+");

        if(conn->file) {
            Database_load(conn);
        }
    }

    if(!conn->file) die("Failed to open the file", conn);

    return conn;
}


void Database_write(struct Connection *conn, int MAX_ROWS, int MAX_DATA)
{
    printf("Enter Database_write\n");
    rewind(conn->file);

    int wc = fwrite(&(conn->db->MAX_ROWS), sizeof(int), 1, conn->file);
    if(wc != 1) die("conn->db->MAX_ROWS Failed to write database.\n", conn);

    wc = fwrite(&(conn->db->MAX_DATA), sizeof(int), 1, conn->file);
    if(wc !=1) die("conn->db->MAX_DATA Failed to write database.\n", conn);

    int i = 0;
    for(i = 0;i < MAX_ROWS; i++){
        fwrite(&((conn->db->rows[i]).id), sizeof(int), 1, conn->file);
        if(wc !=1) die("&((conn->db->rows[i]).id) Failed to write database.\n", conn);
        fwrite(&((conn->db->rows[i]).set), sizeof(int), 1, conn->file);

堆栈泄露case1：

```c
char* name = malloc(sizeof(char) * 20); // 分配20字节的堆内存
name = "";                              // 错误：指针指向字符串常量，丢失堆内存地址
free(name);                             // 错误：试图释放字符串常量，导致未定义行为
```

> #### **错误点**：
>
> 1. **内存泄漏**：
>    - `malloc` 分配的 20 字节堆内存的地址保存在 `name` 中。
>    - 执行 `name = ""` 后，`name` 指向只读的字符串常量 `""`，**原堆内存的地址丢失**，无法再被释放。
> 2. **非法释放**：
>    - `free(name)` 试图释放字符串常量 `""`（位于程序的只读数据段），这是未定义行为（通常导致程序崩溃）。

```c
# 修正后的代码
char* name = malloc(sizeof(char) * 20); // 分配20字节
if (name != NULL) {                     // 检查分配是否成功
    strcpy(name, "");                   // 初始化内容为空字符串
    // 或 strcpy(name, "Hello");        // 也可赋其他值
    free(name);                         // 释放堆内存
}
#-----------------------------------------------------------
char* name = malloc(sizeof(char) * 20);
if (name != NULL) {
    memset(name, 0, 20);                // 全部字节初始化为0（等价于空字符串）
    free(name);
}
```
