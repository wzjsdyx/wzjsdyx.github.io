# SHM格式波形及其dump方式

## 方式一：TCL脚本

```tcl
# Consider a  database  command that opens a default SHM database called  waves .

database -open -shm top_sim -default

# probe -create top_sim -depth all -all -memories

probe -create top_sim                                                          -depth 1
probe -create top_sim.ac7843x                                                  -depth 1
probe -create top_sim.ac7843x.MTCMOS_wrap                                      -depth 1
probe -create top_sim.ac7843x.cm4_core                                         -depth 1

probe -create top_sim.ac7843x.u_flash_whole_top                                -depth 1
probe -create top_sim.ac7843x.u_flash_whole_top.u_pflash_top.u_pf_bank0_state  -depth all -all -memories
probe -create top_sim.ac7843x.u_flash_whole_top.u_dflash_top.u_df_bank1_state  -depth all -all -memories

probe -create top_sim.ac7843x.u_flash_whole_top.u_dflash_top -depth all -all -memories
probe -create top_sim.ac7843x.u_flash_whole_top.u_pflash_top -depth all -all -memories
probe -create top_sim.ac7843x.u_hsm_top_pwr_wrap.hsm_top_pwr_wrap_woiso_u1     -depth all -all -memories
probe -create top_sim.ac7843x.u_standby_ctl_pwr_wrap.standby_ctl_u1            -depth all -all -memories
probe -create top_sim.ac7843x.top_design_ctl                                   -depth all -all -memories
probe -create top_sim.ac7843x.MTCMOS_wrap.MTCMOS.i_i_MTCMOS                    -depth all -all -memories

probe -create top_sim.ac7843x.pad                                              -depth all -all -memories
probe -create top_sim.ac7843x.pinmux                                           -depth all -all -memories
probe -create top_sim.ac7843x.imux                                             -depth all -all -memories

run
```



NOTE:

1. 编译仿真的时候需要添加选项：`+vwdb+shm2vwdb=1`
2. 仿真的是时候添加选项：`-input probe_pattern.tcl`

<font color=red>这种方式也可以将波形文件分开dump ==》TBD</font>



## 方式二：system task

波形文件达到500MB的时候，会自动切换到下一个文件进行存储；

```verilog
initial begin
	$shm_open("top_sim.shm", , , , 500, );
	// $shm_probe(top_sim.ac7843x.cm4_core, "ACMTF", top_sim.ac7843x.MTCMOS_wrap.MTCMOS.i_i_MTCMOS.cm4_MTCMOS.cm4_infra_ctl, "AMTF");
	$shm_probe(top_sim , "ACMTF");
end
```



NOTE:

<font color=blue> 这种dump方式有使用缺陷，只能dump verilog格式文件，无法dump VHDL格式文件</font>





# VWDB格式波形及其dump方式

## 方式一：TCL脚本

```tcl
ida_database -open -name top_sim

ida_probe -wave -wave_probe_args="top_sim                       -all -depth 1 -memories"
ida_probe -wave -wave_probe_args="top_sim.ac7843x               -all -depth 1 -memories"
ida_probe -wave -wave_probe_args="top_sim.ac7843x.MTCMOS_wrap   -all -depth 1 -memories"
ida_probe -wave -wave_probe_args="top_sim.ac7843x.cm4_core      -all -depth 1 -memories"

ida_probe -wave -wave_probe_args="top_sim.ac7843x.u_flash_whole_top.u_dflash_top                            -all -depth all -memories"
ida_probe -wave -wave_probe_args="top_sim.ac7843x.u_flash_whole_top.u_pflash_top                            -all -depth all -memories"
ida_probe -wave -wave_probe_args="top_sim.ac7843x.u_hsm_top_pwr_wrap.hsm_top_pwr_wrap_woiso_u1              -all -depth all -memories"
ida_probe -wave -wave_probe_args="top_sim.ac7843x.u_standby_ctl_pwr_wrap.standby_ctl_u1                     -all -depth all -memories"
ida_probe -wave -wave_probe_args="top_sim.ac7843x.top_design_ctl                                            -all -depth all -memories"
ida_probe -wave -wave_probe_args="top_sim.ac7843x.MTCMOS_wrap.MTCMOS.i_i_MTCMOS                             -all -depth all -memories"

run
```

# 减少波形文件size以及GUI loading的常用方法

1. 使用VWDB格式文件（相比SHM格式，存储占用更小）
2. 设置单个波形文件上限，超过上限之后，自动存储到下一个文件
3. dump hier instance
4. 只在需要的时间段dump波形
