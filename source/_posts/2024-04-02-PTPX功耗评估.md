# step3:PTPX环境配置

1. 将fsdb 放在work/fsdb下面
2. 修改./scr/input_setup.tcl

> set vars(STD_CELL_TRACK)    	 7T     
>
> set vars(netlist)           				 /proj/AH11722A/share/central/backend/pr/data_release/ECO/eco1_20240923/ah11722a/netlist/ah11722a.pr.v 
>
> set vars(sdc_func0p9v)       		/proj/AH11722A/atc3024/ptpx/run_power_script/KITE_norm.sdc 
>
> set vars(design)            			  ah11722a
>
> set vars(design_hier)        		top_sim/u_ah11722a

<font color=blue>总的来说，TCL中的分号用于将多个命令放在同一行时区分命令，不是每条命令都必须使用分号。</font>

<font color=blue>不添加sdc运行PTPX，针对Macro的功耗影响比较大；</font>

3. 修改./work/run.csh

> setenv conditions      	"ffg0p99v150c ffg0p99v125c tt0p9v25c"
> setenv sim_version      "tdi_0809"
> setenv read_fsdb        1

4. source ./work/run.csh
