### **常见颜色设置**

ANSI 颜色代码分为两部分：**前景色（文字颜色）** 和 **背景色**。

| **颜色** | **代码（前景色）** | **代码（背景色）** | **示例**              |
| -------- | ------------------ | ------------------ | --------------------- |
| 黑色     | `30`               | `40`               | `\033[1;30m` 黑色文字 |
| 红色     | `31`               | `41`               | `\033[1;31m` 红色文字 |
| 绿色     | `32`               | `42`               | `\033[1;32m` 绿色文字 |
| 黄色     | `33`               | `43`               | `\033[1;33m` 黄色文字 |
| 蓝色     | `34`               | `44`               | `\033[1;34m` 蓝色文字 |
| 品红色   | `35`               | `45`               | `\033[1;35m` 品红文字 |
| 青色     | `36`               | `46`               | `\033[1;36m` 青色文字 |
| 白色     | `37`               | `47`               | `\033[1;37m` 白色文字 |



### **设置方法示例**

```C
#include <stdio.h>

int main() {
    printf("\033[1;31m This is red text 	\033[0m\n");   	// 红色
    printf("\033[1;32m This is green text 	\033[0m\n"); 	// 绿色
    printf("\033[1;33m This is yellow text 	\033[0m\n"); 	// 黄色
    printf("\033[1;34m This is blue text 	\033[0m\n");   	// 蓝色
    printf("\033[1;35m This is magenta text \033[0m\n");	// 品红
    printf("\033[1;36m This is cyan text 	\033[0m\n");   	// 青色
    return 0;
}
```



### **代码说明**

1. `\033` 或 `\x1B`：
   - ANSI 转义序列的起始符，告诉终端接下来是控制命令。
2. `[1;36m`：
   - `1`：设置文本为高亮（粗体）。
   - `36`：颜色代码，表示青色（cyan）。
3. `*******************************************************`：
   - 打印的实际文本。
4. `\033[0m`：
   - 终止颜色控制，将后续文本恢复到默认颜色
