# 常用操作

## 替换

> 使用`s`，表示substitute
>
> :[range]s/{pattern}/{replace}/[flags]
>
> - `range`：指定要替换的范围
>
> - `pattern`：要查找的字符串或正则表达式
>
> - `replace`：要替换成的字符串
>
> - `flags`：控制替换方式（比如是否全部替换）



### ① 当前行替换一次

```
:s/old/new/
```

→ 把当前行 **第一个** 出现的 “old” 替换成 “new”。

------

### ② 当前行全部替换

```
:s/old/new/g
```

→ 当前行所有 “old” 都换成 “new”。

------

### ③ 全文替换

```
:%s/old/new/g
```

→ 在 **整个文件** 中，把所有 “old” 换成 “new”。

------

### ④ 全文替换并逐个确认

```
:%s/old/new/gc
```

→ 每遇到一个匹配项都会提示：

- `y` → 替换
- `n` → 跳过
- `a` → 全部替换
- `q` → 退出

------

### ⑤ 指定行范围替换

```
:10,20s/old/new/g
```

→ 只在 **第 10 行到第 20 行** 中替换。

------

### ⑥ 忽略大小写

```
:%s/old/new/gi
```

→ 匹配 “old”、“Old”、“OLD” 等
