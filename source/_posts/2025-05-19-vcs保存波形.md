# 保存fsdb波形

1、vcs添加编译选项：

```bash
-debug_access+all+dmptf -debug_pp +memcbk -debug_region=cell+lib -Mupdate
```



2、testbench中添加task语句

> ```verilog
> initial begin
>     $fsdbAutoSwitchDumpfile(500,"kite_execution_tb.fsdb",150);
>     $fsdbDumpvars(0, kite_execution_tb,"+mda");
> end
> ```

# verdi使用vcs生成的中间文件

1、vcs添加编译选项

```bash
-kdb
# 仿真目录下会生成 ./simv.daidir/kdb.elab++/
```



2、verdi加载

```bash
verdi -elab ./simv.daidir/kdb.elab++/ -ssf kite_execution_tb_000.fsdb &
```
