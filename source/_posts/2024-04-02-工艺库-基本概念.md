# 工艺角SS/TT/FF和片上误差OCV

由于半导体制造过程中存在**工艺波动**，相同设计的芯片在不同批次中会有**性能差异**。

- **工艺角**用来表示芯片在**不同制造条件下**的**性能变化范围**。
- 常用的工艺角有：
  - **SS**：Slow-Slow
  - **FF**：Fast-Fast
  - **TT**：Typical-Typical
  - **SF/FS**：Slow-Fast / Fast-Slow（P管和N管速度不匹配）

{% asset_img image-20250403152530125.png %}

同一个批次的wafer其corner是相同的，例如下订单之后，其所有的wafer都是相同的corner；



然后同一片wafer上，不同的位置，其性能有可能也是不同的，这就是OCV的概念；

<font color=blue>OCV的全称是On Chip Variation，用于描述在一块芯片上不同管子间由于工艺偏差、电压降以及温度变化（PVT）等引起的delay变化</font>

# 栅极长度

`HDC50和HDC40`

数字表示栅极长度；

沟道长度越短，速度越快，功耗越高



# Track

 **1. Track的含义**

- 基本定义：
  - 一个Track通常表示一个金属布线层的宽度和间距之和，即布线间隔的单位长度。
  - 单元高度通常是以Track数来表示的，例如8-Track、9-Track、12-Track等。
- 标准单元高度：
  - 单元高度 = Track数 × 单一轨道高度。
  - 一般情况下，1个Track约等于工艺栅极长度（Gate Length），比如40nm工艺中，1个Track大约是40nm。

**4. Track对PPA的影响**

| **Track数** | **面积（Area）**   | **性能（Performance）** | **功耗（Power）** |
| ----------- | ------------------ | ----------------------- | ----------------- |
| **低Track** | **面积小，密度高** | **性能较低**            | **功耗低**        |
| **高Track** | **面积大，密度低** | **性能较高**            | **功耗高**        |

分析：

- 低Track数（如8-Track）：
  - 面积小：标准单元较矮，布局密度高。
  - 性能低：单元较矮，驱动能力受限，不适合高速逻辑。
  - 功耗低：由于短导线和小驱动器，功耗较低。
- 高Track数（如12-Track）：
  - 面积大：标准单元较高，导致芯片面积增大。
  - 性能高：更大的单元高度允许更强的驱动能力，有助于高频电路。
  - 功耗高：驱动能力增强的同时，漏电流和动态功耗增加。
